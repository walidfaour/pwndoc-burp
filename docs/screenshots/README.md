# Screenshots

This directory contains screenshots for documentation.

## Required Screenshots

Please add the following screenshots:

1. `context-menu.png` - Shows the right-click context menu with PwnDoc options
2. `finding-editor.png` - Shows the main finding editor window
3. `cvss-calculator.png` - Shows the CVSS 3.1 calculator section
4. `vuln-library.png` - Shows the vulnerability library panel
5. `install-extension.png` - Shows Burp Suite's extension installation dialog

## Screenshot Guidelines

- Use PNG format for best quality
- Recommended width: 800-1200px
- Crop to show relevant area
- Blur or redact any sensitive information
- Use consistent Burp Suite theme (light recommended for visibility)
